# 架构模式

```text
┌─────────────────┐
│   HTTP Client   │
└─────────────────┘
         │
┌─────────────────┐
│   Gin Router    │  ← API 网关层
└─────────────────┘
         │
┌─────────────────┐
│   Middleware    │  ← 中间件层（认证、日志）
└─────────────────┘
         │
┌─────────────────┐
│   Handler       │  ← 控制器层（HTTP 处理）
└─────────────────┘
         │
┌─────────────────┐
│   Service       │  ← 业务逻辑层
└─────────────────┘
         │
┌─────────────────┐
│   Repository    │  ← 数据访问层
└─────────────────┘
         │
┌─────────────────┐
│   MySQL + GORM  │  ← 数据存储层
└─────────────────┘
```
