核心功能特性

用户系统

注册、登录、JWT认证、个人信息管理

商品系统

商品CRUD、分类管理、搜索筛选、库存管理

购物车系统

添加商品、数量管理、智能合并、实时验证

订单系统

完整下单流程、支付模拟、状态管理、库存扣减

技术栈

Go + Gin
MySQL + GORM
JWT认证
分层架构
Redis缓存
Docker部署

API接口预览

用户相关API

POST /api/v1/register - 用户注册
POST /api/v1/login - 用户登录
GET /api/v1/profile - 获取个人信息

商品相关API

GET /api/v1/products - 获取商品列表
GET /api/v1/products/:id - 获取商品详情
POST /api/v1/products - 创建商品
